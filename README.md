# templ_v3
<p>Gulp project example for website development
</p>

<h2>Описание </h2>
<p>
Сборка gulp для front-end. 
</p>

<h2>Требования</h2>

<ul>

<li> Windows 10 </li>
<li> nmp 3.10.7 </li>
<li> node 8.9.3 </li>
<li> git bach </li>
<li> PowerShell Admin  </li>
<li> bower</li>

</ul>
<h2>Установка</h2>
<ul>
 <ol> bower install (git bash) </ol>
 <ol> npm   install </ol>
</ul>

<h2> Команды</h2>

<ul>
  <li>gulp dev  (основная) </li>
  <li>gulp prod  </li>
</ul>
